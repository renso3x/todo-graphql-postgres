## Node + Typescript + Graphql + Postgres

- CRUD Todo endpoint

## Tech Stack

- Express
- Typescript
- Graphql - (type-graphql)
- Postgres

## Installation

- `npm install`

## Run the application

- `bash scripts/pg.sh` - run a docker pg instance
- `npm run watch` - to compile typescript file
- `npm run dev` - to run dev server
